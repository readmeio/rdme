`rdme help`
===========

Display help for rdme.

* [`rdme help [COMMAND]`](#rdme-help-command)

## `rdme help [COMMAND]`

Display help for rdme.

```
USAGE
  $ rdme help [COMMAND...] [-n]

ARGUMENTS
  COMMAND...  Command to show help for.

FLAGS
  -n, --nested-commands  Include all nested commands in the output.

DESCRIPTION
  Display help for rdme.
```
